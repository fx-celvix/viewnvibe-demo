# **App Name**: Culinary Canvas

## Core Features:

- Restaurant Info Display: Display restaurant information: name, address, contact details.
- Menu Display: Menu display: organized by category (appetizers, entrees, desserts).
- Image Gallery: Image gallery to showcase dishes and restaurant ambiance.
- Recipe Suggestion Tool: AI-powered tool to provide recipe recommendations based on available ingredients. When applicable the LLM may incorporate a recipe or recipes in its reply.
- Customer Reviews: Display user reviews (if available).
- Social Media Links: Link to restaurant's social media profiles.

## Style Guidelines:

- Primary color: Rich burgundy (#800020) evoking sophistication and culinary passion.
- Background color: Off-white (#F5F5DC), providing a neutral and appetizing backdrop.
- Accent color: Golden yellow (#FFD700) for highlighting key elements like prices and calls to action, creating contrast and visual interest.
- Headline font: 'Belleza' sans-serif for headings, creating an elegant visual association to fashion and design. Body font: 'Alegreya' serif, suitable for readability in menus and descriptions.
- Use minimalist line icons to represent food categories and dietary restrictions.
- Clean and spacious layout with a focus on high-quality imagery. The use of whitespace should separate different sections, and sections should correspond to the items listed in 'coreFeatures'.
- Subtle hover effects on menu items and image transitions.
